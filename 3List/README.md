# 第3章 列表

## 3.1 列表的抽象数据类型定义

列表是一组有序的数组。列表中的元素可以是任意数据类型。元素数量受到程序内存限定。

不包含任何元素的列表称为空列表。

列表有描述元素位置的属性。列表有前有后。

列表的抽象数据类型并未指明列表的存储结构，我们使用数组来存储元素。
